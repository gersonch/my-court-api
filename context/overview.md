# Overview

> Resumen del proyecto.

## Qué es

API REST para gestionar complejos deportivos de fútbol 5, 7 y 11. Permite a usuarios reservar canchas, crear tornejos, calificar complejos y más.

## Funcionalidades Principales

- **Gestión de Usuarios**: Registro, login, perfil, roles (user/admin)
- **Complejos Deportivos**: CRUD de complejos con imágenes
- **Canchas**: Gestión de canchas por complejo (tipo, precio, disponibilidad)
- **Reservas**: Sistema de reservas con validación de horarios
- **Torneos**: Creación y gestión de tornejos deportivos
- **Calificaciones**: Sistema de ratings y reviews

## Público Objetivo

- Dueños de complejos deportivos
- Jugadores que buscan canchas para jugar
- Organizadores de tornejos

## Estado

Proyecto en desarrollo activo.
